# color-detection
<h3>Normal HSV</h3>
<ul>
	<li>H 300</li>
	<li>S 100</li>
	<li>V 100</li>
</ul>

<h3>Open CV HSV</h3>
<ul>
	<li>H 180</li>
	<li>S 255</li>
	<li>V 255</li>
</ul>

<h3>Rumus Ubah HSV Normal ke Open CV</h3>
<ul>
	<li>H Open CV = H Normal/2</li>
	<li>S Open CV = S Normal*2.55</li>
	<li>V Open CV = V Normal*2.55</li>
</ul>

<h3>Lower Open CV = H-10, S-20, V-20</h3>
<h3>Upper Open CV =  H+10, S+20, V+20</h3>
